# APT Early Detection — Руководство по запуску

> Дипломный проект: Раннее обнаружение APT-атак по первым N пакетам сетевого потока
> Интерпретация предсказаний: **SHAP** (TreeExplainer)

---

## Структура проекта

```
apt_early_detection/
├── api/              — FastAPI REST API (предсказания в реальном времени)
├── dashboard/        — Streamlit-дашборд (визуализация)
├── data/             — Загрузка и конвертация датасетов
├── dataset/raw_csv/  — CSV-файлы датасетов
│   ├── iscx_combined.csv   (115 446 потоков — основной датасет)
│   ├── iscx_real.csv       (55 446 потоков из ARFF)
│   └── iscx_pcap_early.csv (60 000 потоков из PCAP)
├── features/         — Извлечение признаков из потоков
├── ml/               — Обучение, оценка, SHAP-объяснения
├── models/           — Сохранённые модели (.pkl) и графики
├── realtime/         — Детектор реального времени
├── tests/            — Автотесты (pytest)
├── utils/            — Логирование
├── run_all.py        — Запуск всего пайплайна одной командой
├── Dockerfile
└── docker-compose.yml
```

## Пошаговый запуск

### Шаг 1 — Обучение моделей

```bash
cd C:/Users/Nitro\ V15/ul/apt_early_detection

python -X utf8 -m ml.train_model --csv dataset/raw_csv/iscx_combined.csv
```

### Шаг 3 — SHAP-объяснения

```bash
python -X utf8 -m ml.shap_explain --csv dataset/raw_csv/iscx_combined.csv --samples 6
```

> Время выполнения: ~8 минут на 115 000 строк (TreeExplainer)

**Что сохраняется в `models/`:**
```
shap_beeswarm.png         — глобальный вклад признаков по всему тест-сету
shap_waterfall_1..4.png   — объяснение конкретных потоков (правильные + ошибки)
shap_summary.csv          — таблица mean|SHAP| по каждому признаку
```

**Как читать SHAP-графики:**
- **Beeswarm**: каждая точка — один поток. Красный = высокое значение признака, синий = низкое. Ось X — насколько признак сдвигает предсказание к VPN или Normal.
- **Waterfall**: цепочка решений для одного потока. Синие стрелки → в сторону Normal, красные → в сторону VPN.

**Эндпоинты:**

| Метод | Путь              | Описание                              |
|-------|-------------------|---------------------------------------|
| GET   | `/health`         | Liveness probe                        |
| GET   | `/ready`          | Readiness probe (проверяет модели)    |
| POST  | `/predict/early`  | Классификация по 14 early-признакам   |

**Пример запроса к API:**

```bash
curl -X POST http://localhost:8000/predict/early \
  -H "Content-Type: application/json" \
  -d '{
```

## Признаки (14 early-features)

| Признак           | Описание                                      |
|-------------------|-----------------------------------------------|
| avg_packet_size   | Средний размер пакета (байт)                  |
| std_packet_size   | Стандартное отклонение размера пакетов        |
| min_packet_size   | Минимальный размер пакета                     |
| max_packet_size   | Максимальный размер пакета                    |
| avg_interarrival  | Среднее межпакетное время (с)                 |
| std_interarrival  | Станд. откл. межпакетного времени             |
| min_interarrival  | Минимальное межпакетное время                 |
| max_interarrival  | Максимальное межпакетное время                |
| incoming_ratio    | Доля входящих пакетов (0..1)                  |
| packet_count      | Число пакетов в потоке (обычно 5)             |
| total_bytes       | Суммарный объём (байт)                        |
| flow_duration     | Длительность потока (с)                       |
| bytes_per_second  | Пропускная способность                        |
| pkts_per_second   | Частота пакетов                               |

## Частые проблемы

| Проблема | Решение |
|----------|---------|
| `UnicodeEncodeError` на Windows | Всегда добавляй `-X utf8` |
| `Model 'best_early_model' not loaded` | Сначала запусти `ml.train_model` |
| SHAP очень долго | Нормально — TreeExplainer на 115K строк ~8 мин |
| `streamlit: command not found` | `pip install streamlit` или `python -m streamlit run ...` |
| Порт 8000 занят | `uvicorn api.app:app --port 8001` |
